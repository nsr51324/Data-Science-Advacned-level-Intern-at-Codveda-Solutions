# Data Science Advanced Level Intern at Codveda Solutions

## Overview

This repository contains a collection of Data Science, Machine Learning, Deep Learning, and Natural Language Processing projects completed during the Advanced Level Data Science Internship at Codveda Solutions.

The projects demonstrate practical experience in data preprocessing, exploratory data analysis, machine learning model development, deep learning, natural language processing, and time series analysis.

## Projects

### 1. Feedforward Neural Network (FNN)

A Deep Learning project built with TensorFlow and Keras for handwritten digit recognition using the MNIST dataset.

**Highlights**

* Neural Networks
* TensorFlow & Keras
* Image Classification
* Model Evaluation
* Confusion Matrix Analysis

**Model**

* FNN.h5

### 2. Daily Climate Time Series Data

A Time Series Analysis project focused on exploring and forecasting climate-related variables.

**Highlights**

* Data Cleaning
* Exploratory Data Analysis
* Time Series Visualization
* Predictive Modeling
* Model Serialization

**Model**

* climate_model.pkl

### 3. Text Classification (NLP)

A Natural Language Processing project for text classification using machine learning techniques.

**Highlights**

* Text Preprocessing
* TF-IDF Vectorization
* Classification Models
* NLP Pipeline Development
* Performance Evaluation

**Model**

* text_classifier.pkl
